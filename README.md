#Installation
```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'
inhibit_all_warnings!

target :TargetName do

pod 'Reveal-SDK',:git =>"https://github.com/yidahis/Reveal-SDK-4.git", :configurations => ['Debug']

end
```
Then, run the following command:

```bash
$ pod install
```
